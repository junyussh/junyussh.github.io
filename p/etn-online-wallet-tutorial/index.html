Electroneum 線上錢包教學

上回已經介紹過 ETN CLI 錢包的基本使用教學，今天要介紹的是 ETN 官方出的線上錢包，使用線上錢包，只要有瀏覽器就能輕鬆管理錢包，要轉帳什麼的也不需要輸入一堆指令，與 CLI 錢包相比方便性較高，對於一般使用者也易於使用，畢竟 ETN 是為了方便支付而衍生的貨幣。

註冊

先到 Electroneum 的線上錢包官網(https://my.electroneum.com/)註冊一個帳戶，點「Create Account」進入註冊頁面。

可以使用 Facebook 登入，或著用 Email 來註冊。

接著到你的信箱收驗證信，點裡面的網址。

然後會問你的手機電話號碼，會傳簡訊驗證，最後會要你輸入五位數的 PIN 碼，這個 PIN 碼很重要，付款登入都需要，忘記就麻煩了。

登入

驗證完手機號碼就能登入線上錢包了，到登入頁面輸入帳密後，要再輸入剛才註冊輸的 PIN 碼，所以說很重要。

注意：登入時 PIN 碼只能嘗試三次。

進入線上錢包後，會看見以下畫面，載入過程較為緩慢，需稍等。

載入完成的畫面大概長這樣，界面還不錯看。Recent Transactions 是最近的交易紀錄；Wallet Balance 是錢包餘額；右下方有 QR Code 的地方是這個線上錢包的地址，別人若要轉帳給你，直接匯到這個地址就可以入到線上錢包。

如果英文看不習慣，右上角有切換語言的功能，有中文的選項喔，不過是簡體中文。

從離線錢包匯入

官方錢包還提供一項功能是把離線錢包的錢轉移到線上錢包中。

在旁邊的 Wallet Balance 點「Import from Offline Wallet」。

第一步要輸入錢包地址，用掃描 QR Code 的方式也可以。

然後要分別輸入 Private View Key、Private Spend Key。

如果之前是用線上產生紙錢包的話，PDF 檔裡面都有這些資訊。

用 [CLI 錢包產生](https://junyussh.gitlab.io/p/etn-wallet-generate/#CLI 錢包)的話，在錢包裡輸入 viewkey、 spendkey，記得填 secret 的部分。

其實只是幫你把錢包裡所有的錢轉過去，還是會扣手續費 1 ETN，自己用 CLI 錢包轉也可以。

手機 APP

Electroneum 官方也有推出手機版 APP，可與線上錢包連結，用手機也能方便轉帳，不過目前只有出 Android 版本，iOS 可能還得要等一陣子。

Android: https://play.google.com/store/apps/details?id=com.electroneum.mobile&hl=zh-TW

不知道怎麼寫的，這 APP 載入有點緩慢，跟網頁版一樣要等一會。

登入要輸入 PIN 碼。

如果手機語言是設定中文，初次進入應該是簡體中文界面，如果不喜歡可以在「更多」的地方改語言。

抽免費 ETN

目前因為在推廣手機 APP，所以官方舉辦了 ETN 的抽獎活動，每周會送出 5000 ETN 的獎金，在「更多」的地方有個「贏取免費幣」，在英文版叫做 “Earn FREE Coins”。

輸入推薦碼就可以參加抽獎活動了，如果需要的話不妨填入我的推薦碼：5CDD4B。
