使用 LAME 輕鬆將無損 wav 壓制成 MP3 格式

2018-09-02
教學 Linux 影音處理
約 593 字 預計閱讀 2 分鐘

文章目錄

    • 安裝
      • Windows
      • Linux
    • 壓制

Photo by Dmitry Bayer on Unsplash

LAME 是一套開源的 MP3 Encoder，可以將音源壓制成標準的 MP3 格式，一般擷取 CD 裡的音源大部分都是無損的 wav 格式，檔案大小大，不好攜帶，今天來教大家如何使用 LAME ，輕鬆的將無損音樂壓制成 mp3 格式。

安裝

Windows

這次用到的工具需要 LAME 編碼器，如果沒有此檔案的人請先下載。

如果你有安裝小丸工具箱，在程式的 tools 目錄也有此工具。

這裡提供已經編譯好的 LAME 3.100 給 Windows 平台使用者下載，此載點由 LAME 官方推薦，若你的 Windows 版本低於 XP 就要用其他的編譯版本(我想應該是不會有)。

  • 64 位元下載：按這裡
  • 32 位元下載：按這裡

Linux

Arch Linux:

1

sudo pacman -S lame

Debian or Ubuntu:

1

sudo apt-get install lame

Redhat, Fedora or CentOS:

1

sudo yum install lame

壓制

把 lame.exe 解壓縮後在同個目錄執行 CMD，準備下指令。

小技巧：可以在資料夾路徑的地方輸入 cmd 就會在這個資料夾下啟動 CMD 了。

下面指令可以取得 LAME 命令列使用說明。

1

lame --help

將原來的無損音檔為 music.wav 壓製成 music.mp3。

1

lame music.wav music.mp3 -b 192

-b 參數設定 MP3 碼率，碼率愈高，音質愈好，最小 65，最高可設 320，平常我都追求最高品質，用 320 kbps 壓制。

LAME 在 Linux 作業系統下也是下相同的指令。

如果有裝 foobar2000 播放器的話可以直接透過 foobar2000 介面進行轉換。

先將欲壓制的音源開啟，按右鍵選取 轉換 > 快速轉換。

接著選取 MP3 格式，按下「轉換」，若為第一次使用此功能會出現對話視窗，要求選取 lame.exe 路徑。

壓制可能會用上一點時間，視檔案大小決定。

經過實際測試，接近 700 MB 的 wav，用 320 kbps 可以被壓制成大概 150 MB 的 mp3，壓縮率頗高的，不過我聽起來是沒有什麼差異。
