新環境，新氣象

2017-10-17
日常隨筆
約 456 字 預計閱讀 1 分鐘

文章目錄

話說先前我本想遠端用 Hexo 寫作，於是把 Hexo 的原始檔案都放在了 Coding.net 的 WebIDE 了，最近想發表文章卻無法進入 WebIDE，發 Issue 也沒有解決問題，這下只能下大絕了——重新建置 Hexo。

沒錯，所有的原始檔都遺失了，不過繼續放在 WebIDE 也不是什麼辦法， 還好我大部分的文章的 Markdown 都放置在 Cmd Markdown 裡，稍作增益便可重新發佈。

{% quote 蔣經國, 故總統 %} 今天應該做的事，今天不做，明天才做，你會永遠後悔。 {% endquote %}

重新安裝一次 Hexo 真是折騰，不過能一勞永逸也算值得。

佈景主題也從原本的 Light 暫時換成了 Apollo，選擇 Apollo 的原因乃其 UI 簡潔，這樣版面看起來清爽多了，閱讀起來也比較舒服，再經過一番修改，終至上架，如果以後閒閒沒事再來自己設計佈景主題。

Hello World 的日期依舊我當初架站的日期，之後也會回復一些舊文章，日期一樣為最初的發稿日期，若看到以前的文章忽然出現請別見怪。

junyussh.github.io 舊站，但未來有可能會更新

託管也從 Github Pages 換到 GitLab Pages。

換到 GitLab 有三大優點：

  1. 支援上傳 SSL 證書
  2. 免費私密 Git 倉庫
  3. 自動集成功能(GitLab CI)

第三點才是最重要的，這樣才能自動產生靜態檔並傳到倉庫上，之後可能會詳細介紹。
