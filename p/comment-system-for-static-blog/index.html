靜態部落格留言板蒐集

2017-12-12
留言板 筆記
約 603 字 預計閱讀 2 分鐘

最近想為自己的 Hexo Blog 增加留言功能，一般架設靜態部落格所使用的留言板都是 Disqus 居多，網路上的教學也一堆，不過之前爆出 Disqus 五年前被駭，我就想說除了 Disqus，我們還有其他的選擇嗎？答案是肯定的，以下是我從網路上所蒐集的留言版服務，不過有些是需要 Github 才能使用。

Comm(ent|it)

將留言變成 commit，用 JavaScript 腳本提交至 Github 倉庫，可以用 Twitter、Facebook 或 Github 登入。

https://commentit.io/

來必利

來自南韓的一款靜態部落格留言版，類似 Disqus，支援各社群平台登入，缺點是無法匿名留言(不過其他的似乎也不行)。

http://www.laibili.com.cn/

HyperComments

https://www.hypercomments.com/

介紹文：从今天开始使用HyperComments啦！ | 闲停

Valine

一款快速、简洁且高效的无后端评论系统。

需要註冊 LeanCloud 取得 API Key 才能使用。

https://valine.js.org/#/

介紹文：hexo博客评论新神器——Valine | GISerDaiShaoqing's Blog

Gitalk

需要 Github 帳戶登入，用 Github Issue + Preact 開發而成，支援多種語言，並自動判斷用戶當前語言，還支援 Markdown 語法。

https://gitalk.github.io/

介紹文：善用 Github Issue 的開源評論插件 – Gitalk | Calpa's Blog

Any else?

還有友言及搜狐暢言等留言版，不過是大陸公司開發的，介面用著不習慣，而且網站需要備案。

Consequence

怕麻煩的還是乖乖用 Disqus 就好了，大部分的佈景主題都能方便添加 Disqus 留言板，而且 Disqus 使用者多，介面體驗優良、社群活躍，說明文件也較詳細，生態也較良好。

上述留言板如果要使用，我會推薦 Comm(ent|it)、HyperComments 和 Gitalk，這些用起來會比較高端。

以後也許會自己造個輪子吧。
